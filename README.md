# MyURL_Scanner.py
Scan URLs in online sandboxes directly from the command line
Automated, secure, and fast analysis using the public urlscan.io API.

Overview

URLSandboxCLI is a command-line tool that allows you to submit URLs for online sandbox analysis using the urlscan.io public API.

It is designed for:

Security analysts

OSINT professionals

Ethical hackers and pentesters

Incident response teams

Automated URL analysis workflows

The tool does not execute malware locally and does not interact with websites beyond sending the URL to the remote sandbox. It simply retrieves and displays the final report.

Features

Submit a URL to an online sandbox with a single command

Automatic polling to wait for the final analysis result

Display key information extracted from the report:

Risk level

Contacted domains

Screenshot generated by the sandbox

Detected technologies (when available)

Export results to JSON

Bulk mode for analyzing multiple URLs from a file

Simple configuration via environment variables

Compatible with Linux, macOS, and Windows

Installation

Requirements:

Python 3.9+

1. Clone the repository
git clone https://github.com/TeoVMP/URLSandboxCLI

cd URLSandboxCLI

2. Install dependencies
pip install -r requirements.txt

cd URLSandboxCLI

2. Install dependencies
pip install -r requirements.txt

3. Install the tool
pip install -e .

API Key Setup

Create an account at: https://urlscan.io/apikey/

Then set your API key:

Linux / macOS
export URLSCAN_API_KEY="YOUR_API_KEY_HERE"

Windows (PowerShell)
setx URLSCAN_API_KEY "YOUR_API_KEY_HERE"

Usage
Scan a single URL
urlsandbox scan https://example.com


Example output:

Submitting URL: https://example.com
Scan ID: 20240205-abc123
Waiting for results...

Risk: low
Contacted domains:
 - example.com
 - cdn.example.net

Screenshot: https://urlscan.io/screenshots/20240205-abc123.png

Bulk mode (multiple URLs)

File urls.txt:

https://example.com
https://test-site.net
https://sample-domain.xyz


Execute:

urlsandbox bulk urls.txt --out results.json


Output:

[+] Scanning https://example.com
[+] Scanning https://test-site.net
[+] Scanning https://sample-domain.xyz
Results saved to results.json
